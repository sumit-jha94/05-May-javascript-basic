Syntax is rules to write any code

let a = 7; - this is javascript code

This whole process of writing a code is called as "Declaring Variables"

a is called as - identifier

= is called as assignment operator

7 is literal


    Rules Of Chosing variables names:-

      1. letters, digit, underscore and $ sign
      2. Must begin with a, $, _, letter
      3. This is case sensitive
      4. Javascript reserved keywords cannot be used.
      5. Cannot start with number - 1,2--

Const Vs Let Vs Var

var - used before ES6 code. It could cause error in our code. Need to be avoided.

var can be updated and re-declared.


let - used to declare block scoped variables.

let can be updated not re-declared.


const - is used to declare constant. The values cannot be changed throughout our course.

const can neither be redeclared nor updated.

const cannot be changed. We have to take other varible name.
