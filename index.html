For Loops

We use loops to perform repeated actions.

Types Of Loop In JS

  1. For Loop - loop a block of code no. of times.
  2. For-In Loop - Loops through the keys of Object.
  3. For Of - Loops through the values of an object.
  4. While - Loops a block based on specific condition.
  5. Do-While - While loop variant which runs atleast once.

for(statement 1; statement 2; statement 3){

//Code to be executed

}

    Statement
  1. Statement 1 is executed 1 time.
  2. Statement 2 is the conditions on which the loop runs.
  3. Stament 3 is executed everywhere the loop body is executed.

Strings is collections of character.

Strings are used to store and manipulate text.

Strings can be created as below
  1. let name = "Sumit"
  2. let name = 'Sumit'

    Template Liberals

It uses backticks insteads of quote to define a strings.

let name = `sumit`

With template literals, we can use single and doble quote inside a string.

let sentence = `The name "is" sumit's dairy`

We can directly insert variabel into template literals. This is called string interpolartion

let a = `This is ${name}`

name is variable

    Escape Sequence Character

If we have to print below, JS will misunderstood the code.

let name = 'Adam D'Angelo'

We can use single quote escape to solve this issue.

let name = 'Adam D\'Angelo'

Similarly we can use \" inside a string with double quote.

Other Escape Sequence Character are as follow:-

  1. \n - Newline
  2. \t - Tab
  3. \r - Return
